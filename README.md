# devops-capstone-project

Customer Accounts Microservice for the IBM DevOps Capstone Project.

This project implements a RESTful Account Management Microservice for an e-commerce platform. The service manages customer account information and provides APIs to create, read, update, delete, and list customer accounts. The application is developed using Flask, PostgreSQL, Docker, Kubernetes, GitHub Actions, and Tekton CI/CD pipelines following DevOps best practices.

## Features

* Create customer accounts
* Read customer account details
* Update existing customer accounts
* Delete customer accounts
* List all customer accounts
* Containerized using Docker
* Deployed to Kubernetes/OpenShift
* Automated CI/CD with GitHub Actions and Tekton
* Test-driven development with automated unit testing

## Repository

This repository contains the source code, tests, deployment manifests, and CI/CD pipeline definitions required to complete the IBM DevOps Capstone Project.
